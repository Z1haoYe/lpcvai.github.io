Track 1

Image classification for different lighting conditions and styles

The participants will classify images in real-world situations, where conditions are often unpredictable and uncertain. Each team will receive 9,000 images, with 1,000 images for each class (a combination of Indoor/Outdoor/Synthetic under low light, normal light, and backlit conditions). The dataset will be based on COCO object detection classes, which contain 80 different classes. The score is defined as the ratio of accuracy and execution time. To receive a score of a classified image, the computer program must return the correct label of the object in the image.
